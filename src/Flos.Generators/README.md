# Flos.Generators

Source generators for compile-time code generation. Eliminates runtime reflection for AOT-safe operation.

## Installation

```xml
<PackageReference Include="Flos.Generators" />
```

## Generators

### DeepClone Generator

Generates `IDeepCloneable<T>.DeepClone()` implementations for `IStateSlice` types annotated with `[DeepClone]`. Clones all instance properties and fields regardless of accessibility.

```csharp
[DeepClone]
public class GameState : IStateSlice, IDeepCloneable<GameState>
{
    public int Score { get; set; }
    public IOrderedMap<string, int> Inventory { get; set; }
    public IOrderedSet<string> Tags { get; set; }
    // DeepClone() is auto-generated
}
```

**Supported field types:**

| Type | Clone strategy |
|------|---------------|
| Value types (`int`, `float`, structs, enums) | Direct copy |
| `string` | Direct copy (immutable) |
| `IOrderedMap<K, V>` | Copied; values deep-cloned if they implement `IDeepCloneable<T>` |
| `IOrderedSet<T>` | Copied (elements must be value types or strings) |
| `List<T>` | Copied; elements deep-cloned if they implement `IDeepCloneable<T>` |
| `Dictionary<K, V>` | Copied; values deep-cloned if they implement `IDeepCloneable<T>` |
| `HashSet<T>` | Copied (elements must be value types or strings) |
| `T[]` (arrays) | Copied; elements deep-cloned if they implement `IDeepCloneable<T>` |
| `IDeepCloneable<T>` | Calls `DeepClone()` recursively |

Unsupported reference types trigger **FLOSGEN001** at compile time.

**Generated code example** (conceptual):

```csharp
// Generated by Flos.Generators
public GameState DeepClone()
{
    var clone = new GameState();
    clone.Score = this.Score;                          // DirectCopy
    clone.Inventory = CloneOrderedMap(this.Inventory); // OrderedMapCopy
    clone.Tags = CloneOrderedSet(this.Tags);           // OrderedSetCopy
    return clone;
}
```

### TypeResolver Generator

Produces a `FlosTypeResolver` mapping deterministic integer Type IDs to concrete types for polymorphic serialization. Hash collisions are detected at compile time (FLOSGEN002).

### Registration Generator

Generates handler and applier registration code for types marked with `[AutoRegister]`. Produces a static `FlosRegistration.RegisterAll(IHandlerRegistry)` method.

Types **without** `[AutoRegister]` are not included and must be registered manually — this allows constructor parameters and custom wiring.

```csharp
// Auto-registered (parameterless constructor required):
[AutoRegister]
public class AttackHandler : ICommandHandler<AttackCmd>
{
    public ErrorCode Handle(AttackCmd cmd, IStateReader state, EventBuffer events) => default;
}

// Manually registered (constructor injection allowed):
public class DamageHandler : ICommandHandler<DamageCmd>
{
    private readonly DamageConfig _config;
    public DamageHandler(DamageConfig config) => _config = config;
    public ErrorCode Handle(DamageCmd cmd, IStateReader state, EventBuffer events) => default;
}

// In your module's OnInitialize:
registry.Register(new DamageHandler(myConfig));
```

## Diagnostics

| ID | Severity | Description |
|----|----------|-------------|
| FLOSGEN001 | Error | Non-cloneable reference field in `IStateSlice` |
| FLOSGEN002 | Error | TypeResolver hash collision between two types |
| FLOSGEN003 | Warning | `[DeepClone]` on type that doesn't implement `IStateSlice` |
| FLOSGEN004 | Error | `[AutoRegister]` on handler/applier type without a parameterless constructor |
